## What is a Formal Concept? ü§î

A **formal concept** is a pair (A, B) where:
* **A** is a subset of objects called the **extent**.
* **B** is a subset of attributes called the **intent**.

This pair must satisfy two conditions:
1.  The intent **B** consists of exactly those attributes shared by all objects in the extent **A**.
2.  The extent **A** consists of exactly those objects that possess all attributes in the intent **B**.

To find these, we use **derivation operators** (denoted by a prime symbol ' ):
* For a set of objects $A \subseteq G$, its derivation $A'$ is the set of all attributes common to all objects in A:
    $A' = \{m \in M \mid \forall g \in A, (g, m) \in I\}$
* For a set of attributes $B \subseteq M$, its derivation $B'$ is the set of all objects that possess all attributes in B:
    $B' = {g \in G \mid \forall m \in B, (g, m) \in I}$

So, a pair $(A, B)$ is a formal concept if $A' = B$ and $B' = A$. This means that $A$ is the set of all objects that have all attributes in $B$, and $B$ is the set of all attributes shared by all objects in $A$.
