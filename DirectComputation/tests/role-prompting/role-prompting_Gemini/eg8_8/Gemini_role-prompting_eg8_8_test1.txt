## What's Formal Concept Analysis?

At its heart, FCA is a mathematical theory that helps us find meaningful groupings in data. It starts with a **formal context**, which is basically a table that shows us which objects have which attributes.

A formal context is a triplet $(G, M, I)$, where:
* $G$ is a set of **objects**.
* $M$ is a set of **attributes**.
* $I$ is a binary relation between $G$ and $M$, where $(g, m) \in I$ means "object $g$ has attribute $m$."

From a context, we can derive **formal concepts**. A formal concept is a pair $(A, B)$ where:
* $A \subseteq G$ is a set of objects called the **extent**.
* $B \subseteq M$ is a set of attributes called the **intent**.
The key idea is that:
1.  All objects in $A$ share *all* the attributes in $B$.
2.  There are no other objects outside of $A$ that share *all* attributes in $B$.
3.  There are no other attributes outside of $B$ that are shared by *all* objects in $A$.

To find these concepts, we use derivation operators:
* For a set of objects $A \subseteq G$, its intent $A'$ is defined as:
    $A' = \{m \in M \mid \forall g \in A, (g,m) \in I\}$ (all attributes common to objects in $A$).
* For a set of attributes $B \subseteq M$, its extent $B'$ is defined as:
    $B' = \{g \in G \mid \forall m \in B, (g,m) \in I\}$ (all objects possessing attributes in $B$).

A pair $(A, B)$ is a formal concept if $A' = B$ and $B' = A$. This is equivalent to saying $A'' = A$ (where $A'' = (A')'$) and $B = A'$, or $B'' = B$ and $A = B'$.
