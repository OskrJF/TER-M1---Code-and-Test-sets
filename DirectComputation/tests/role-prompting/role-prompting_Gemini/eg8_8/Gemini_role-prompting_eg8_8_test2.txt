**How do we find these?**

We systematically look for these pairs. For any set of objects $A$, we find all attributes $A'$ common to them. Then, we find all objects $(A')'$ that have all attributes in $A'$. If $(A')' = A$, then $(A, A')$ is a formal concept. Similarly, we can start with a set of attributes.

For example, to get **Concept 2**:
* Start with object O1. Its attributes are {a1, a2, a3}[cite: 1]. Let this be our initial Intent.
* Find all objects that have {a1, a2, a3}: These are O1, O2, O3, O6[cite: 1]. This is our Extent.

* Since the Extent {O1, O2, O3, O6} yields the Intent {a1, a2, a3}, and this Intent yields back the same Extent, we have found a formal concept: ({O1, O2, O3, O6}, {a1, a2, a3}).
