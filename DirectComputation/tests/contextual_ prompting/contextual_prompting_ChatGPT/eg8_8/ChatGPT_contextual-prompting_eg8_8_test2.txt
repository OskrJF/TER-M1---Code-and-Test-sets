### ðŸ”¹ Step 2: Define Concepts

A **formal concept** is a pair (extent, intent) such that:

* The **extent** is a set of objects sharing all attributes in the intent.
* The **intent** is the set of attributes common to all objects in the extent.

We extract all such pairs where:

* Extent = objects sharing a given set of attributes.
* Intent = attributes shared by a given set of objects.

You can derive these using a simple method such as **attribute closure**:

* Pick a set of objects.
* Find the common attributes (intent).
* Then find all objects that share those attributes (extent).
* If you get back the same set of objects, it forms a valid concept.
