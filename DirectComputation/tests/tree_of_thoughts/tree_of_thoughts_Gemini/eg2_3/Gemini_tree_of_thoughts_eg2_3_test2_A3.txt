    Manual Iteration and Closure: For a small context, you can start by systematically examining individual objects or attributes. For each object, identify all attributes it possesses to form an initial intent. Then, find all other objects that share this exact set of attributes; this group of objects forms the extent. The pair (extent, intent) is a formal concept. Similarly, you can start with an attribute, find all objects possessing it (initial extent), and then identify all attributes common to these objects (intent).
